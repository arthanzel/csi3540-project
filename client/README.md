Quantum of Cells SPA Client
===========================

Building
--------
Make sure that `NODE_ENV` is not `production`.

    npm install
    cp .env.defaults .env
    npm start
    
This runs a development server at `http://lvh.me:8080`. Running this development server is necessary for authentication to work.
